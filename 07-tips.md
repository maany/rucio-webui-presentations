# Keyboard Shortcuts

<div class="grid grid-cols-2 gap-8">

<div>

## Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Cmd/Ctrl + /` | Show keyboard shortcuts |
| `Esc` | Close modals/dialogs |
| `Tab` | Navigate form fields |
| `Arrow Keys` | Navigate lists |

</div>

<div>

## Navigation Tips

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Click` | Open in new tab |
| `Alt + Arrow` | Browser back/forward |
| `Home/End` | Top/bottom of page |
| `/` | Focus search |

</div>

</div>

<v-click>

**Pro Tip:** Press `Cmd/Ctrl + /` anytime to see all available shortcuts!

</v-click>

# Productivity Tips

<v-clicks>

## Work Smarter

### 1. **Use Bookmarks Effectively**
- Bookmark your daily-use datasets and storage elements
- Keep your HotBar organized and current
- Remove outdated bookmarks regularly

### 2. **Master the Command Palette**
- Learn to search by partial names
- Use it for navigation instead of clicking
- Discover new features through search

### 3. **Leverage Filters**
- Use filters to narrow down large result sets
- Save common filter combinations
- Combine multiple filters for precise results

### 4. **Keyboard Navigation**
- Learn essential keyboard shortcuts
- Stay in "keyboard mode" for efficiency
- Reduce context switching to mouse

</v-clicks>

# Common Workflows

<div class="grid grid-cols-2 gap-8">

<div>

## Finding a Dataset

<v-clicks>

1. Press `Cmd/Ctrl + K`
2. Type dataset name
3. Select from results
4. View details and metadata

**Or:** Use filters on the datasets page

</v-clicks>

## Creating a Rule

<v-clicks>

1. Navigate to your dataset
2. Click "Create Rule"
3. Select destination RSE
4. Set copies and lifetime
5. Submit and monitor

</v-clicks>

</div>

<div>

## Monitoring Subscriptions

<v-clicks>

1. Go to Subscriptions page
2. Filter by account or status
3. Bookmark active subscriptions
4. Check regularly for updates

</v-clicks>

## Checking Storage

<v-clicks>

1. Open Command Palette
2. Search for RSE name
3. View capacity and usage
4. Monitor availability

</v-clicks>

</div>

</div>

# Accessibility Best Practices

<v-clicks>

## For All Users

### Screen Reader Users
- Full keyboard navigation support
- Semantic HTML structure
- ARIA labels on interactive elements
- Status announcements for dynamic content

### Visual Accessibility
- High contrast mode available
- Zoom support up to 200%
- Color is not the only indicator
- Clear focus indicators

### Keyboard-Only Users
- All features accessible via keyboard
- Logical tab order
- Skip links for navigation
- Visible focus states

### Cognitive Accessibility
- Clear, consistent interface
- Helpful error messages
- Progress indicators for long operations
- Undo/cancel options available

</v-clicks>
