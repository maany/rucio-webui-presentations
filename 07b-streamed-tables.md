# What are Streamed Tables?

Advanced table system for efficiently handling large datasets:

- **Real-Time Streaming** - Data loads progressively
- **Dynamic Filtering** - Filter by any column
- **Column Sorting** - Sort data ascending/descending
- **Resizable Columns** - Adjust widths as needed
- **Pagination** - Navigate large datasets easily
