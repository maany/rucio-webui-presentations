# What is Command Palette?

## Quick Access to Everything


A powerful search interface that lets you:

- <span class="i-heroicons-outline-magnifying-glass w-5 h-5"/> **Search** - Find any resource instantly
- <span class="i-heroicons-outline-rocket-launch w-5 h-5"/> **Navigate** - Jump to any page or feature
- <span class="i-heroicons-outline-pencil-square w-5 h-5"/> **Discover** - Explore available commands

**Access:** Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux)

# How to Use Command Palette

<div class="grid grid-cols-2 gap-8">

<div>

## Opening the Palette


- **Keyboard:** `Cmd/Ctrl + K`
- **Click:** Search icon in navigation bar
- **Anytime, Anywhere** in the application


## What You Can Do

- Search for datasets by name
- Navigate to specific pages
- Jump to storage elements

</div>

<div>

## Supported Search Types

- **DIDs** - Search by dataset name or pattern
- **RSEs** - Find storage elements
- **Rules** - Look up replication rules
- **Pages** - Navigate to any section
