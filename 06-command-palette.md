# What is Command Palette?

## Quick Access to Everything

<v-clicks>

A powerful search interface that lets you:

- <span class="i-heroicons-outline-magnifying-glass w-5 h-5"/> **Search** - Find any resource instantly
- <span class="i-heroicons-outline-rocket-launch w-5 h-5"/> **Navigate** - Jump to any page or feature
- <span class="i-heroicons-outline-bolt w-5 h-5"/> **Execute** - Run actions without clicking
- <span class="i-heroicons-outline-pencil-square w-5 h-5"/> **Discover** - Explore available commands

**Access:** Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux)

</v-clicks>

::right::

## Why It Matters

<v-clicks>

- **Faster** than clicking through menus
- **More efficient** than remembering URLs
- **Keyboard-friendly** for power users
- **Discoverable** - learn new features
- **Context-aware** - relevant suggestions

</v-clicks>

# How to Use Command Palette

<div class="grid grid-cols-2 gap-8">

<div>

## Opening the Palette

<v-clicks>

- **Keyboard:** `Cmd/Ctrl + K`
- **Click:** Search icon in navigation bar
- **Anytime, Anywhere** in the application

</v-clicks>

## What You Can Do

<v-clicks>

- Search for datasets by name
- Navigate to specific pages
- Jump to storage elements
- Access account settings
- Execute quick actions

</v-clicks>

</div>

<div>

## Search Tips

<v-clicks>

- **Be specific** - Type partial names
- **Use filters** - Narrow by type
- **Recent items** - Show first
- **Fuzzy matching** - Typos are OK
- **Keyboard navigation** - Arrow keys to select

</v-clicks>

## Examples

```
"data16" → Find datasets
"settings" → Open settings
"CERN" → Storage at CERN
```

## Smart Search

- **Instant Results** - As you type
- **Relevance Ranking** - Best matches first
- **Category Grouping** - Organized by type
